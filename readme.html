このファイルについて

H-IIB HTV MODをOrbiter2016に対応させるパッチです。
D3D9Clientを導入していない環境では利用できません。

HTV（こうのとり）MODのダウンロード・インストール - Home for Orbinauts

インストール

ConfigとTexturesの2つのフォルダをコピーする。
Orbiterのインストールフォルダにペーストして上書き。

LaunchpadでVideo→Advancedをクリック。
Enable terrain flatteningを有効にする。

使用方法

Orbiterを起動する。
LaunchpadでJAXAフォルダをダブルクリック。
JAXA launchシナリオでゲームを開始する。

キーボードでPを押すと、30秒後に自動打ち上げを開始する。
軌道投入が終了したら、Jを押してHTVを分離する。

他の2つのシナリオは、必要なファイルが公開停止になっているので利用できない。

ロケットの高さを調整する

環境によっては、ロケットと発射台の高さにずれが生じることがある。

Config\JAXA\H2B.iniを、メモ帳などのテキストエディタで開く。
cog=の数字を変更すると、ロケットの地上からの高さを変更できる。
